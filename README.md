# FireEscapeRoom

A Unity-based escape room game project.

## Project Overview

FireEscapeRoom is a Unity game project that includes gameplay scripts, scenes, audio, models, and supporting project settings.

## Requirements

- Unity Editor (version matching this project)
- Windows (current development environment)
- .NET/Visual Studio tooling (for C# script editing and solution builds)

## Getting Started

1. Open Unity Hub.
2. Add this folder as a project:
   - `d:/Games/My Own/FireEscapeRoom`
3. Open the project in Unity.
4. Open a scene from `Assets/Scenes/`.
5. Press Play in the Unity Editor.

## Build From Editor

1. In Unity, open **File > Build Settings**.
2. Select your target platform.
3. Add the desired scenes to **Scenes In Build**.
4. Click **Build**.

## Build From VS Code Task

This workspace includes a `build` task that runs:

- `msbuild /property:GenerateFullPaths=true /t:build /consoleloggerparameters:NoSummary`

Use the VS Code Task runner to execute it.

## Key Project Folders

- `Assets/` - Game content and scripts
- `Assets/Scripts/` - C# gameplay and systems code
- `Assets/Scenes/` - Unity scenes
- `Packages/` - Unity package dependencies
- `ProjectSettings/` - Unity project configuration
- `UserSettings/` - Local editor/user settings

## Notes

- The generated `.csproj` and `.sln` files are Unity-managed and can be regenerated by Unity.
- The `Library/` folder contains local cache/build artifacts and is machine-specific.
